# WordGames

In this application, words are given randomly and you derive new words that can be formed in English from that word. If you write the given word, write a word with three letters or less, if you want to retype the same word and try to create a word that is not in English, you will get an error. If the word is correct, it will be added to your list. You can use the words repeatedly with the Refresh button.
